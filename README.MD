## Development

To run frontend only:

```bash
npm run hot
```

To run both:

```bash
// terminal 1
npm run dev

// terminal 2
npm run hot
```

## Production

Frontend:

```bash
npm run build:frontend
```

Backend:

```bash
npm run start
```
